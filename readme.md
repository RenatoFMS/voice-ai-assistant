# Assistente de Voz: Programação com GPT-4 

Este projeto é um assistente virtual capaz de ouvir comandos de voz, processá-los com IA e responder por áudio, focado no contexto de desenvolvimento de software.

##  Créditos
Este projeto foi desenvolvido com base no excelente artigo da **DIO (Digital Innovation One)**:
- [Conversando por voz com o ChatGPT utilizando Whisper, OpenAI e Python](https://web.dio.me/articles/conversando-por-voz-com-o-chatgpt-utilizando-whisper-openai-e-python)

##  Tecnologias Utilizadas
- **OpenAI Whisper:** Para transcrição de áudio em texto.
- **OpenAI GPT-4:** Para processamento de linguagem natural com foco em programação.
- **gTTS (Google Text-to-Speech):** Para a síntese da voz de resposta.
- **Google Colab:** Ambiente de execução.

##  Como usar
1. Instale as dependências: `pip install -r requirements.txt`
2. Configure sua `OPENAI_API_KEY` nas variáveis de ambiente.
3. Execute o script `main.py`.
